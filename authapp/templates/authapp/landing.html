How It Works

Simple, secure,
seamless

1

Add the Script

Copy the script tag and paste it into any HTML page you want to protect.

2

Visitor Enters Email

When someone visits your page, they see a verification overlay asking for their email.

3

OTP Sent via Email

A 6-digit verification code is emailed instantly. Valid for only 30 seconds.

4

Access Granted

Enter the correct code and the page reveals itself. Wrong code? Stay locked out with a clear error.

Integration

Add to your site
in 10 seconds

Copy this single line and paste it into the <head> or before </body> of any HTML page.

HTML 
<script src="/static/js/2fa-guard.js"></script>
💡
That's it! No API keys, no configuration, no backend setup. The script automatically handles the entire verification flow.

Example Integration

index.html 
<!DOCTYPE html>
<html>
<head>
  <title>My Protected Page</title>

  <!-- Add this one line to protect your page -->
  <script src="/static/js/2fa-guard.js"></script>
</head>
<body>
  <h1>Secret Content</h1>
  <p>This page is protected by email OTP verification.</p>
</body>
</html>
API

REST API
for custom flows

Want more control? Use the API directly to build custom verification flows.

POST
/api/send-otp/

Send a one-time verification code to an email address.

Request 
{ "email": "user@example.com" }
Response 
{ "token": "abc123...", "message": "OTP sent" }
POST
/api/verify-otp/

Verify the OTP entered by the user.

Request 
{
  "email": "user@example.com",
  "otp": "123456",
  "token": "abc123..."
}
Response (Success) 
{ "verified": true }
